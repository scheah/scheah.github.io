Smartwatch Gesture Authentication – Android Wear & Machine Learning

  • With the growing popularity of smartwatches and other wearable technology, effort has been put into seeing if these devices can be used as a new means of authentication.
  • Using accelerometer 3-dimensional data sampled over a window of time from Android Wear, model features from data and utilize machine learning techniques to train a classifier.
  • Aimed to implement an activity and user identification model on a smartwatch and use that model as a second authentication step for added security without additional user overhead
